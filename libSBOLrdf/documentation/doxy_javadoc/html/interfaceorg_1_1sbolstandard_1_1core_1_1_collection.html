Detailed Description

The SBOL data model's Collection for RDF and Json.

The objects of this type represent an organizational container which helps users and developers conceptualize a set of DNA components and SequenceFeatures as a group. Any combination of these objects can be added to a Collection, annotated with a displayID, name and description and be shared on the web. Such collections could be a set of restriction enzyme recognition sites, such as the features commonly used for BBF RFC 10 BioBricks. A Collection could contain all the DNA components used in a specific project, lab, or any custom grouping specified by the user.
